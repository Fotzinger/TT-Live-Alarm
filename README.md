# 🎥 TT Live Alarm Bot

🚀 **Never miss a TikTok livestream again!**
This Telegram bot notifies you when selected TikTok streamers go live.

## 🔥 Features

* 🟢 Notifications when a streamer goes live
* 🧠 Prevention of duplicate alerts
* 📲 Control via Telegram (`/add`, `/remove`, `/list`)
* 💾 Persistent storage of tracked streamers

---

## 📲 Bot Commands

```text
/start
```

Start the bot

```text
/add username
```

Add a TikTok streamer

```text
/remove username
```

Remove a streamer

```text
/list
```

Show all tracked streamers

---

## 🧠 How it works

* The bot periodically checks livestream status
* When a stream is detected, you receive a notification
* Short interruptions are ignored
* Small delays may occur
